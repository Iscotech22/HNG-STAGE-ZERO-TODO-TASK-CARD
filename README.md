# 📝 Todo Task Card

A clean, modern, and accessible Todo/Task Card built with HTML, CSS, and JavaScript.

This project focuses on **testability, accessibility, and responsiveness** across devices.

## ✨ Features

- ✅ Semantic HTML structure
- ✅ Fully responsive design (320px – 1200px)
- ✅ Accessible (ARIA labels, keyboard navigation)
- ✅ Real-time time remaining calculation
- ✅ Completion toggle (checkbox)
- ✅ Status update (Pending → Done)
- ✅ Tags/Categories support
- ✅ Edit & Delete buttons (dummy actions)
- ✅ LocalStorage support (persists data after refresh)

---

## 🧪 Testability

All required elements include `data-testid` attributes for automated testing:

- `test-todo-card`
- `test-todo-title`
- `test-todo-description`
- `test-todo-priority`
- `test-todo-due-date`
- `test-todo-time-remaining`
- `test-todo-status`
- `test-todo-complete-toggle`
- `test-todo-tags`
- `test-todo-edit-button`
- `test-todo-delete-button`

---

## ♿ Accessibility

- Proper semantic elements (`section`, `h2`, `p`, `time`, `button`)
- `aria-label` for icon-only buttons
- Keyboard navigable (Tab support)
- Visible focus states
- Good color contrast

---

## 📱 Responsiveness

- Mobile-first approach
- Works across:
  - 320px (small phones)
  - Tablets
  - Desktop (up to 1200px)
- Flexible layout using Flexbox
- No horizontal overflow

---

## ⚙️ Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (DOM, LocalStorage)
